安装GradleDependencyHelper插件

IntellJ IDEA仓库中的插件（github）

  1. 安装

    • 需要在File->Settings->Plugins菜单中选择Browse Repository…,如下图：

    • 然后搜索GradleDependencyHelper，点击Install

  2. 使用

    • 直接在.gradle文件中输入ArtifactId不会自动补全
    • 需要打开Tools->GDependency菜单，然后输入关键词，搜索
    • 找到需要的ArtifactId，双击后选择需要的版本，然后copy，在Ctrl+V到.gradle文件中

  3. 总结

    相对于直接去maven仓库中搜索，这种使用方法对于开发人员来说并没有方便多少

JetBrains Plugins Repository中的插件(github)

  1. 安装
    由于这个插件在IntellJ IDEA中没有搜索到，所以去官网下载插件，下载与上边一样，只不过安装时选择Install plugin from disk.直接选中GradleDependenciesHelperPlugin-..jar文件安装
  2. 使用
    直接在.gradle文件中输入你需要依赖的ArtifactId的名称，按shift+ctrl+space会自动提示补全

  3. 总结

    这个方法与我们平时编程写代码时的自动补全习惯类似，可以加快我们程序猿的开发速度
