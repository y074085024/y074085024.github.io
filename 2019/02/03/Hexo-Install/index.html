hexo介绍

hexo是一个快速、简洁且高效的博客框架

虚拟机安装

  1. 使用virtualbox创建虚拟机，类型选择Linux，版本选择Red Hat（64-bit）
    图片
  2. 进入centos官网下载最新的64位iso镜像，我下载的是CentOS-7-x86_64-Minimal-1810.iso
  3. 网络选择桥接模式，光驱中选择刚下载的iso文件，启动虚拟机，直接安装，时间选择一下其余默认，并为root用户设置一个密码
    图片
  4. 编辑网卡配置文件

  3. 设置静态地址（可选）

hexo安装

  1. 安装hexo

如果安装时卡住，先在~/.nvm/.npmrc文件中添加
registry = http://registry.cnpmjs.org
再进行安装，如果失败可能是网络原因，重试安装
